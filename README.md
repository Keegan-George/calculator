# Calculator 
A simple calculator built using Javascript, HTML, and CSS. Currently supports the basic arithmetic operations: addition, subtraction, multiplication, and division. 

This is the final project of the [Odin Project's Foundations Course](https://www.theodinproject.com/paths/foundations). It is a culmination of everything learned to this point, with intention of testing your understanding of core web development concepts. 

## Instructions
1. Clone the repository to your local machine
2. Open the index.html file in your browser.
